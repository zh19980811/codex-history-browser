Codex History Quick Migration

This folder contains two one-click scripts:
- export.cmd: Package Codex history into codex-history-export.zip
- import.cmd: Extract codex-history-export.zip into the target .codex folder

Usage
1) On source machine: double-click export.cmd
2) Copy codex-history-export.zip to target machine
3) On target machine: double-click import.cmd

Notes
- Close Codex before running import.
- The scripts auto-detect %USERPROFILE%\.codex
